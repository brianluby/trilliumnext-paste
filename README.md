# TrilliumNext Paste

An Alfred Workflow for macOS that allows you to quickly capture notes and ideas to [TrilliumNext](https://github.com/TriliumNext/Notes).

## Quick Start

1. Download the [latest release](https://github.com/brianluby/trilliumnext-paste/releases)
2. Double-click `TrilliumNext Paste.alfredworkflow` to install
3. Configure your TrilliumNext URL and ETAPI token
4. Use `tn` in Alfred to capture notes
